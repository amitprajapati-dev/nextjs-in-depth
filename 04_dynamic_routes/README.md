# 🚀 Dynamic Routes

Dynamic Routes allow us to create pages whose URL contains dynamic values.

Instead of creating separate pages for every item, we can use a single route to handle multiple URLs.

For example:

- /products/1
- /products/2
- /products/100

All these URLs can be handled by a single page.

# 🤔 Why?

Why use Dynamic Routes?

Without Dynamic Routes, you would need a separate page for every product, user, or blog post.

Dynamic Routes allow one page to handle many different URLs.

---

# 💡 Interview Tip

### Q. What is a Dynamic Route?

A Dynamic Route is a route that contains dynamic values, allowing a single page to serve multiple URLs using route parameters.
