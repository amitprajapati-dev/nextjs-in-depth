# 🔗 Link Component in Next.js

Navigation is one of the most common tasks in every web application.

Next.js provides a built-in `Link` component that allows users to navigate between pages without performing a full page reload.

Unlike a normal HTML `<a>` tag, the `Link` component enables faster navigation, preserves the application state, and improves the overall user experience.

# 🤔 Why?

### Why use the Link component?

The Link component provides faster navigation by avoiding full page reloads and improving the user experience.

---

# 💡 Interview Tip

### Q. Why should we use `<Link>` instead of `<a>`?

**Answer:**

`<Link>` enables client-side navigation, making page transitions faster without reloading the entire application.
