# 🚀 Nested Dynamic Routes

Nested Dynamic Routes allow us to create multiple dynamic segments in a single URL.

Instead of handling only one dynamic value, we can capture multiple values from the URL.

Example:

- /products/1/reviews/10
- /users/amit/posts/5
- /courses/react/lectures/3

---

# 🎯 Learning Goals

- Understand nested dynamic routes
- Capture multiple route parameters
- Build real-world URL structures

---

# 💻 Practice

We'll build:

- Product Reviews
- User Posts

---

# 🤔 Why?

Many real-world applications require multiple dynamic values in a URL.

Example:

A product has many reviews.

Instead of creating separate pages, one route can handle all product reviews.

### Q. What are Nested Dynamic Routes?

Nested Dynamic Routes allow multiple dynamic parameters in a URL by nesting dynamic folders.
