# 📘 Next.js Fundamentals

Welcome to the **Next.js Fundamentals** section.

This section is designed to build a strong foundation before writing any Next.js code. Instead of memorizing concepts, the goal is to understand how Next.js works, why it exists, and when to use it.

A solid understanding of these fundamentals will make advanced topics like Routing, Server Components, Data Fetching, Rendering, Middleware, and Authentication much easier to learn.

---

## 🎯 Learning Goals

By the end of this section, you will be able to:

- Understand what Next.js is
- Explain why Next.js was created
- Understand how Next.js works internally
- Learn the core features of Next.js
- Understand different rendering strategies
- Understand the Next.js project structure
- Compare App Router and Pages Router
- Know when to use Next.js in real-world projects
- Answer common Next.js interview questions with confidence
