# 📍 usePathname in Next.js

The `usePathname()` hook allows us to access the current URL pathname.

It is commonly used to highlight active navigation links, conditionally render UI, and determine the current route.

# 🤔 Why?

The application often needs to know which page is currently active.

Examples:

- Active Navbar
- Active Sidebar
- Breadcrumb Navigation

# 💡 Interview Tip

### Q. What is `usePathname()`?

`usePathname()` is a Client Component hook that returns the current URL pathname.
