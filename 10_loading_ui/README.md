# ⏳ Loading UI in Next.js

Loading UI improves the user experience by displaying temporary content while a page or data is loading.

Instead of showing a blank screen, Next.js allows us to display a loading state automatically using a special `loading.tsx` file.

# 🤔 Why?

### Why do we need Loading UI?

Fetching data takes time.

Without a loading state, users may think the application has stopped working.

A Loading UI provides instant feedback and improves the overall user experience.

---

# 💡 Interview Tip

### Q. What is `loading.tsx` in Next.js?

**Answer:**

`loading.tsx` is a special file that displays a loading UI while a route or its data is being loaded.
