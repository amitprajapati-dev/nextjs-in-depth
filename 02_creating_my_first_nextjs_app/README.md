# 🤔 Why?

### Why use `npx`?

`npx` allows us to execute packages without installing them globally.

---

### Why use `create-next-app`?

It is the official tool provided by the Next.js team to create a new project with the recommended configuration.

---

### Why use the App Router?

The App Router is the modern routing system introduced in Next.js 13 and is recommended for all new projects.

---

### Why use TypeScript?

TypeScript improves code quality by providing type safety, better autocompletion, and easier maintenance.

---

### Why use ESLint?

ESLint helps identify potential errors and enforces consistent coding standards.

---

### Why use Tailwind CSS?

Tailwind CSS enables rapid UI development with utility classes and is widely used in modern React and Next.js projects.

---

# 💡 Interview Tips

### Q. What is `create-next-app`?

**Answer:**

`create-next-app` is the official CLI tool used to quickly create a new Next.js project with the recommended configuration.

---

### Q. What is `npx`?

**Answer:**

`npx` executes npm packages without requiring a global installation.

---

### Q. Why should you use the App Router?

**Answer:**

The App Router provides modern features like Server Components, nested layouts, loading UI, better data fetching, and improved performance.
