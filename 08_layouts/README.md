# 🧩 Layouts in Next.js

Layouts are one of the most powerful features of the Next.js App Router.

A layout allows you to share common UI across multiple pages without repeating the same code.

For example, a Navbar, Sidebar, or Footer can be placed inside a layout so that every page automatically displays them.

# 🤔 Why?

### Why do we use Layouts?

Without layouts, you would need to copy the same Navbar, Sidebar, and Footer into every page.

Layouts allow us to write that code once and reuse it across multiple pages.

This keeps the project clean, maintainable, and easier to scale.

---

# 💡 Interview Tip

### Q. What is a Layout in Next.js?

**Answer:**

A Layout is a special component that wraps one or more pages with shared UI such as a Navbar, Sidebar, or Footer. It helps avoid duplicate code and provides a consistent user interface across multiple pages.
