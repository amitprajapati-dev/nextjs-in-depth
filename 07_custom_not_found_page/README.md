# 🚀 Custom Not Found Page

A **Not Found Page (404 Page)** is displayed when a user visits a route that does not exist.

Instead of showing a generic browser error, Next.js allows us to create a custom 404 page that provides a better user experience.

# 🤔 Why?

### Why create a custom 404 page?

A good 404 page helps users understand that the requested page doesn't exist and allows them to navigate back instead of seeing a confusing error.

---

# 💡 Interview Tip

### Q. What is `not-found.tsx` in Next.js?

**Answer:**

`not-found.tsx` is a special file used to display a custom 404 page whenever a route is not found or when the `notFound()` function is called.
